======
myblog
======

My blog made with Pelican, published at http://glenfant.github.io

Build environment
=================

Create and activate a virtualenv

.. code:: console

   pip install -r requirements.txt

Build HTML site
===============

.. code:: console

   make html

Publish the site
================

.. code:: console

   git push origin master
   make github
